# HST Modern — Профессиональный тренажёр азбуки Морзе (PWA)

**HST Modern** — это современное веб-приложение для обучения и тренировки приема азбуки Морзе. Программа разработана с учетом требований скоростной телеграфии (HST) и оптимизирована для работы на любых устройствах: от компьютеров до смартфонов.

## 🌟 Основные возможности

### 1. Четыре режима работы
- **Тренировка (Training):** Классический приём групп по 5 знаков.
- **Обучение (Learning):** Позволяет выбирать конкретные символы для отработки (например, только проблемные буквы).
- **Русские буквы (Russian Mode):** Специальный режим для работы с кириллицей, включая специфические знаки (Я, Ш, Ю, Ч, Э, Ь, Ы).
- **Соревнование (Competition):** Режим «на выносливость». Скорость автоматически увеличивается на 10 зн/мин после каждого успешно принятого текста. Результаты сохраняются в лог-файл.

### 2. Гибкие настройки звука и таймингов
- **Тон (Pitch):** Настройка частоты звука от 300 до 1000 Гц.
- **Скорость:** Плавная регулировка от 20 до 400 знаков в минуту.
- **Паузы:** Индивидуальная настройка пауз между знаками и между группами (в длительностях точек).

### 3. Полная мобильная адаптивность
- Специальная верстка для **горизонтального (Landscape)** и **вертикального (Portrait)** режимов телефона.
- Группы текста всегда умещаются в ширину экрана без горизонтальной прокрутки.
- Кнопка «СТАРТ» закреплена в нижней части для удобного управления большим пальцем.

### 4. Работа в качестве приложения (PWA)
Приложение поддерживает технологию Progressive Web App:
- **Установка на рабочий стол:** Появляется иконка приложения.
- **Офлайн-режим:** Работает без подключения к интернету.
- **Standalone:** Открывается в собственном окне без адресной строки браузера.

## 🛠 Состав проекта (Файлы)

Для корректной работы приложения в папке должны находиться 4 файла:
1.  **`index.html`** — основной файл с интерфейсом и логикой (бывший `hst_modern.html`).
2.  **`manifest.json`** — конфигурация для установки приложения на телефон.
3.  **`sw.js`** — Service Worker для кэширования и работы без интернета.
4.  **`icon.png`** — векторная иконка приложения высокого качества.

## 📱 Инструкция по установке на телефон

После того как вы открыли приложение по ссылке (через HTTPS):

### Android (Google Chrome)
1. Нажмите на три точки **(⋮)** в правом верхнем углу.
2. Выберите пункт **«Установить приложение»** или **«Добавить на главный экран»**.

### iOS (Safari)
1. Нажмите на иконку **«Поделиться»** (квадрат со стрелкой вверх) в нижней панели.
2. Прокрутите список вниз и выберите **«На экран «Домой»»**.

## ⚙️ Технические детали
- **Audio Engine:** Использует Web Audio API для чистого синтеза звука без щелчков.
- **Шрифт:** Используется современный шрифт 'Inter' (системный).
- **Сетка:** CSS Grid с динамическим расчетом масштаба через `clamp` и `vw`.
